## ✨ Features

- ✅ Display list of books
- ✅ Add new book
- ✅ Remove books
- ✅ SPA-style navigation (no URL change)
- ✅ Data persistence with `localStorage`
- ✅ Contact section

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Luxon (for date/time)
- GitHub Actions (CI linting)
- ESLint
